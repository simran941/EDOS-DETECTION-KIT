# EDoS Security Dashboard - Backend

FastAPI backend for real-time cybersecurity monitoring with Supabase integration.

## Quick Start

```bash
# Clone and run (that's it!)
git clone <your-repo>
cd edos-detection-dashboard/backend
uv run python main.py
```

## Development

```bash
# Add new dependencies
uv add package-name

# Run with auto-reload
uv run uvicorn main:app --reload

# Sync dependencies (after git pull)
uv sync
```
